Aplicación de Prueba – Reto Técnico Fullstack

Esta aplicación fue desarrollada como parte de un reto técnico. Permite registrar datos simples de usuario (nombre y edad), almacenarlos en Azure Blob Storage y consumir imágenes públicas o privadas mediante APIs expuestas desde .NET 8. Incluye generación de URL firmadas con SAS para acceso temporal seguro.
